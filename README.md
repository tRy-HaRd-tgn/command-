# Command Plus

Веб-приложение для управления проектами и оценки навыков разработчиков. Платформа позволяет пользователям регистрироваться, проходить тестирование hard skills и soft skills, создавать и управлять проектами.

## Технологии

- **React 18.3.1** - библиотека для создания пользовательского интерфейса
- **Vite 5.4.10** - инструмент сборки и разработки
- **React Router DOM 6.28.0** - маршрутизация в приложении
- **Redux 5.0.1** - управление состоянием приложения
- **React Hook Form 7.55.0** - работа с формами
- **Axios 1.8.1** - HTTP клиент для работы с API
- **Sass 1.81.0** - препроцессор CSS для стилизации
- **ESLint** - линтер для проверки кода

## Установка и запуск

### Требования

- Node.js (версия 16 или выше)
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для production

```bash
npm run build
```

### Предпросмотр production сборки

```bash
npm run preview
```

### Линтинг кода

```bash
npm run lint
```

## Основной функционал

### Аутентификация

- Регистрация новых пользователей
- Вход в систему
- Подтверждение email
- Восстановление пароля

### Профиль пользователя

- Заполнение личных данных
- Настройка профиля (две страницы профиля)
- Управление информацией о себе

### Тестирование навыков

- **Hard Skills** - тестирование технических навыков
- **Soft Skills** - оценка личностных качеств и коммуникативных навыков

### Управление проектами

- Просмотр доступных проектов
- Создание запросов на участие в проектах
- Управление собственными проектами
- Просмотр проектов пользователя

### Дополнительные страницы

- Страница контактов
- Стартовая страница с информацией о платформе

## Маршрутизация

### Публичные маршруты

- `/` - стартовая страница
- `/auth/new-verification` - подтверждение email
- `/auth/new-password` - восстановление пароля

### Приватные маршруты

- `/profile1` - первая страница профиля
- `/profile2` - вторая страница профиля
- `/projects` - страница проектов
- `/projectRequest` - запрос проекта
- `/userProjects` - проекты пользователя
- `/contact` - страница контактов

## Конфигурация

### Vite

Настройки сервера разработки находятся в `vite.config.js`. По умолчанию приложение запускается на порту 5173 с поддержкой следующих хостов:

- localhost
- commandplus-frontend-5gddei-a786c7-95-174-103-115.traefik.me
- sitetrialbox.ru
- command-plus.tech

### Переменные окружения

Для работы с API необходимо создать файл `.env` в корне проекта и указать базовый URL API:

```
VITE_API_URL=your_api_url_here
```

## Разработка

### Стилизация

Проект использует SCSS модули для стилизации компонентов. Каждый компонент имеет свой файл стилей `styles.module.scss`.

### Управление состоянием

Состояние приложения управляется через Redux. Основные редьюсеры:

- `authReducer` - состояние аутентификации
- `userReducer` - данные пользователя

### HTTP запросы

Все HTTP запросы выполняются через настроенный Axios клиент в `src/http/index.js`. Сервисы для работы с API находятся в директории `src/service/`.
